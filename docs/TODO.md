## 📌 프로젝트 개요

### 🦺 보람안전물산(주) 안전용품 홍보 홈페이지

이 프로젝트는 안전용품 전문 유통업체를 위한 **제품 홍보용 웹사이트**입니다.  
실제 클라이언트를 위한 작업으로, **심플하고 직관적인 UI**, **반응형 구성**, **마케팅 목적의 시각적 구성**을 중심으로 개발되었습니다.

### ✨ 완성된 주요 기능
- ✅ 메인 Hero 섹션: 브랜드 메시지 + 대표 이미지
- ✅ 제품 카탈로그: 11개 카테고리, 254개 제품 데이터베이스
- ✅ 카테고리별 제품 목록 및 상세 페이지
- ✅ 통합 검색 시스템 (실시간 제안, 고급 필터링)
- ✅ 관리자 시스템 (제품/카테고리 CRUD)
- ✅ 회사 소개 페이지
- ✅ 모바일/태블릿 완전 반응형 지원
- ✅ Docker 환경 구축 완료

### 🛠 사용 기술
- **Frontend**: Next.js 14 + TypeScript + Tailwind CSS
- **Backend**: FastAPI + SQLAlchemy + PostgreSQL
- **Infrastructure**: Docker + Docker Compose
- **Database**: PostgreSQL 13 (254개 제품, 11개 카테고리)

### 📊 현재 상태
- **개발 완료도**: 약 90%
- **핵심 기능**: 100% 완료 ✅
- **Docker 환경**: 100% 완료 ✅
- **배포 준비도**: 문서화 필요
