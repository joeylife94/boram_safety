## 🎓 사용 방법 (친구에게 설명할 내용)

### Step 1: 실행
```bash
docker-compose up -d
docker-compose exec backend python create_tables.py
```

### Step 2: 회사 정보 설정
```
http://localhost:3000/admin/settings
```
- 회사명을 친구 회사 이름으로 변경
- 연락처, 주소 등 입력
- 저장

### Step 3: 제품 업로드
```
http://localhost:3000/admin/excel
```
- 템플릿 다운로드
- Excel에 제품 데이터 입력
- 업로드

### Step 4: 완료!
```
http://localhost:3000
```
- 친구의 안전용품 회사 웹사이트 완성!

## 🔒 보안 고려사항

### 현재 상태
- Admin 페이지에 인증 없음 (전시 사이트이므로)
- 데이터베이스 비밀번호는 환경 변수로 관리
- SQL Injection 방지 (SQLAlchemy ORM)

### 필요시 추가 가능
```python
# Nginx Basic Auth로 Admin 보호
location /admin {
    auth_basic "Admin Area";
    auth_basic_user_file /etc/nginx/.htpasswd;
}
```
